Local POC - image to Phaser

A compiler for generated game media.

openRender turns raw generated images into Phaser assets, helper code, local reports, and rollbackable project changes. No account, hosted worker, or model provider is required for v0.1.

The local loop

The first workflow stays intentionally small: initialize a project, compile one generated image, then inspect the local report.

01

Initialize

Create config and local run state inside the target project.

openrender init --target phaser --framework vite
02

Compile and install

Validate the source image, create artifacts, and write Phaser helper files.

openrender compile sprite --from tmp/slime_raw.png --id enemy.slime.idle --frames 6 --frame-size 64x64 --install
03

Review or rollback

Open the HTML report, keep the files, or restore the previous state.

openrender report --run latest --open

Visible files, not hidden magic

openRender is a compiler layer for agents. Its work is inspectable: generated assets, TypeScript helpers, run JSON, report HTML, and rollback snapshots stay in the local project.

Built for agentic game development

The page should make the infrastructure position clear: openRender does not generate art. It standardizes the step after generation.

Contract

Convert source media into an explicit visual contract before writing engine files.

visual.sprite_frame_set

Adapter

Generate Phaser-ready file paths, manifest entries, and animation helpers.

target: phaser + vite

Report

Leave a local HTML and JSON trail that a human or coding agent can inspect.

.openrender/reports

Narrow v0.1 scope

The POC proves one reliable path before expanding into more engines, media types, hosted APIs, or MCP surfaces.

Included

Vite + Phaser, image assets, transparent sprites, sprite frame sets, reports, verify, and rollback.

Excluded

Accounts, billing, cloud workers, hosted playgrounds, model calls, audio, video, and marketplace flows.

Positioning

A local media-to-engine compiler that Codex, Cursor, Claude Code, or a human developer can call.
